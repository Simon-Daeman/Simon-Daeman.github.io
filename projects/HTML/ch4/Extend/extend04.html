RGB shows how much red, green, and blue is in a color. Hexadecimal converts the base 10 numbers from rbg into base 16 number from (0-F). HSL shows how much color is used and the shade of the colors.

Preferred color code format: I prefer hexadecimal, because there is less to type when displaying colors than rgb

HWB summary: This is used to more easily understand HSL by using cylindrical-coordination to represent colors

CMYK summary: This method is used in printing and utilizes four colors instead of three; those being cyan, magenta, yellow, and key(black).
