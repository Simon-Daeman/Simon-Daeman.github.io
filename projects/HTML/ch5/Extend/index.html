Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

You have to call the position property, use a colon, then declare the property as "sticky".

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

I used "40px" for the top value and "35px" for the left value and it minimally, yet noticably affected the blank space between the edges of the screen and the box.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

I used "90px" for the top value and "15px" for the right value and the changes present were a lot more drastic.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used "10px" for the bottom value and "30px" for the right value and the box stayed on one part of the screen with minimal changes position-wise.
